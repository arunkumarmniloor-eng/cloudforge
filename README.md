# CloudForge – Cloud Native DevOps Automation Platform

CloudForge is a production-style DevOps project demonstrating how modern cloud-native applications are built, deployed, and monitored using DevOps tools.

This project implements an end-to-end CI/CD pipeline with container orchestration and monitoring using AWS Free Tier resources.

---

## Architecture

Developer → GitHub → Jenkins CI/CD → Docker → DockerHub → Kubernetes (K3s) → Application Deployment → Monitoring (Prometheus + Grafana)

---

## Tools Used

- AWS (EC2)
- Terraform
- Docker
- Kubernetes (K3s)
- Jenkins
- Prometheus
- Grafana
- GitHub

---

## Project Features

- Infrastructure provisioning using Terraform
- Containerized applications using Docker
- Kubernetes-based deployment
- Automated CI/CD pipeline with Jenkins
- Monitoring using Prometheus and Grafana
- AWS Free Tier optimized architecture
